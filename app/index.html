Travelling Teamform

A real-time online team forming app specialized for travelling



Enter a name (e.g. Japan(1/1 - 7/1) to create/join an trip



What is travelling teamform?
A real-time online team forming tool designed for users to create or join a trip plan and allowed users to find members with some same preferences

The following roles are supported in travelling teamform:

Administrator

  • Be responsible for creating the trip plan for team forming (e.g. Tokyo(1/1/2016-6/1/2016) , Seoul(5/5/2016-12/5/2016))
  • Be able to set the minimum and maximum team size for the trip
  • Be able to make sure the details of the planning of the trip are clear

Team

  • Be able to add and remove requests made by members
  • Once a member joins the team, all his/her requests will be deleted

Member

  • Be able to request to join multiple teams
